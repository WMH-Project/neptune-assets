# Photo credits — `photos/expeditions/`

All photographs in this directory are credited to:

> **© Prince Hussain Aga Khan / Focused on Nature**

This single credit applies to **every file** in `photos/expeditions/`. It must
appear visibly anywhere these photos are displayed (web, print, social media,
press kit). The recommended caption is:

```
© Prince Hussain Aga Khan / Focused on Nature
```

## Filename convention

Files follow the pattern `<YYYY-MM><LOC>-<camera-id>[-<process-tags>].jpg`,
preserving the original Focused on Nature catalog identifier:

- **YYYY-MM** — capture year/month (some files only carry the year)
- **LOC** — 3-letter location code (cf. `../ASSETS.md` for the full table)
- **camera-id** — original camera file identifier (e.g. `0r8a6273`,
  `b62a7678`) — kept verbatim so a photo can always be traced back to the
  FON catalogue
- **process-tags** — post-production indicators (`rec`, `shl`, `mike`,
  `v2`, `ok`, etc.) — kept for the same traceability reason

Names are lowercased and spaces/underscores replaced with hyphens, so URLs
stay clean (`a-b-c.jpg` rather than `A_B C.jpg`).
