## Tokens email (simplifié)

```css
--color-primary: #1D2B5C;    /* Deep Navy — brand */
--color-accent: #2E747E;     /* Abyss Teal — secondary */

--color-bg: #F5F0E6;         /* Pearl — fond clair */
--color-highlight: #D8EBFA;  /* Sky — highlights */

--color-text-light: #FFFFFF; /* sur fond navy */
```

## Typographie

- **Titres (display)** : `'Playfair Display', Georgia, 'Times New Roman', serif` (weight 700)
- **Body** : `'Inter', -apple-system, BlinkMacSystemFont, sans-serif` (weight 400)
- **Alternative body** : `'Montserrat', sans-serif` (si cohérence avec Newsletter_01 nécessaire)

## Design principles (extraits du PDF)

- **Expressive** : Organic & Fluid
- **Contemporary** & Minimal
- **Humanist feel** : Resonant & Emotional
- **Mineral** : naturalness

> "A visual language that mirrors the Ocean itself — fluid and luminous, layered & transparent — embodying universality & life."

## Thèmes visuels

1. **Surface** — The First Glance : réflexion, textures iridescentes, jeu de lumière
2. **Depth** — Discovery : gradients, transparences superposées, motifs data-inspired
3. **Horizon** — Action & Cooperation : design ouvert, lumineux, transparent, lignes convergentes
4. **Origin** — The Shared Ground : formes circulaires, continuité, universalité
